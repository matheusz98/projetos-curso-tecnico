The Game Awards

The Game Awards (TGA) é uma cerimônia de premiação anual que reconhece os melhores jogos eletrônicos de cada ano, além de grandes feitos na indústria. Novos títulos também são anunciados durante as cerimônias, que são exclusivamente produzidas e apresentadas por Geoff Keighley. Nesta página serão listados todos os games que venceram a categoria Game Of The Year (Jogo do ano) em seus respectivos anos.
